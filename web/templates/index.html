"STIMA" Route Finder

Aplikasi berbasis web untuk mencari rute berdasarkan file input


Cara menggunakan aplikasi

  1. Upload file matriks dengan format sebagai berikut:

    Pastikan formatnya sudah benar dan tidak ada spasi pada akhir baris

    Keterangan:

    • Baris pertama adalah jumlah simpul
    • Isikan koordinat simpul pada baris kedua hingga selesai
    • Isikan matriks ketetanggaan dengan jarak antar dua simpul yang berhubungan

  2. Klik tombol "Baca File

  3. Pilih lokasi awal dan tujuan, sesuai dengan nama pada lokasi pada file matriks

  4. Klik tombol "Cari Rute"

  5. Aplikasi akan mengarahkan Anda ke halaman hasil
